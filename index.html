Modelo de clasificación de reseñas de películas 🎬

Modelo de Machine Learning 🤖 que clasifica automáticamente las reseñas de películas 🍿 en positivas o negativas 🚫✅, utilizando un conjunto de datos de IMDB 📊. A través de un análisis exploratorio de datos 🔍 y técnicas de preprocesamiento de texto 🧹, se entrenan varios modelos para alcanzar una precisión F1 superior a 0.85 📈, optimizando la clasificación de críticas y facilitando la filtración automática de contenido 📚. Este proyecto pone en práctica técnicas de clasificación de texto y análisis de sentimientos para construir un sistema automatizado para filtrar críticas de películas 🎥.
